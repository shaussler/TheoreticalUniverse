Differential as a Covector

Differential as a Covector#

\[dx^i ∂_j = \frac{∂x^i}{∂x^j} = δ^i_j\]
\[dx^i \frac{∂}{∂ x^j} x^l = dx^i δ_j^l\]
\[\frac{∂x^i}{∂x^j} x^l= δ^i_j x^l\]

Considering a parametric 3surface in a space that can be of higher dimension:

\[f(x, y, z) = C\]

where \(C\) is a constant and therefore \(dC=0\) The differential is:

\[df = dx \frac{∂f}{∂x} + dy \frac{∂f}{∂y} + dz \frac{∂f}{∂z} = 0\]

That we rewrite with

\[df = dx \frac{∂}{∂x} f + dy \frac{∂}{∂y} f + dz \frac{∂}{∂z} f\]
\[df = \left(dx \frac{∂}{∂x} + dy \frac{∂}{∂y} + dz \frac{∂}{∂z}\right) f\]

Considering \(f(x, y, z)=x\):

\[dx = dx \frac{∂}{∂x} x + dy \frac{∂}{∂y} x + dz \frac{∂}{∂z} x\]

And thus:

\[dφ(x) = dx \frac{∂}{∂x} φ(x)\]
\[dφ(x) = α dx\]

Taking the taylor serie of \(φ(x)\):

\[φ(x) = φ(a) + x \frac{∂}{∂x} φ(x) + ...\]

We get:

\[dφ(x) = dx \frac{∂}{∂x} [φ(a) + x \frac{∂}{∂x} φ(x)]\]

Then

\[dφ(x) = dx \frac{∂}{∂x} [x α]\]
\[dx = dx \frac{∂}{∂x} x\]
\[dx \frac{∂}{∂x} = 1\]
\[dx \frac{∂}{∂y} = 0\]
\[dx \frac{∂}{∂z} = 0\]
\[dx^i ∂_j = δ^i_j\]
